**15.1.4 异或运算的一些特性**

* 一个数和自己做异或的结果是 `0`
* 和 `0` 做异或保持原值不变，和 `1` 做异或得到原值的相反值
* 奇偶校验
* `x^x^y==y(x^x==0, 0^y==y)`

### 15.2 其他运算符

**15.2.4 sizeof 运算符与 typedef 类型声明**

* `sizeof 表达式`: 求子表达式的类型, 然后把这个类型所占的字节数作为表达式的值
* `sizeof(类型名)`: 这种类型所占的字节数
